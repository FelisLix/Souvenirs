Сувеніри

Інтерфейс для додатка "Сувеніри" створено за допомогою JavaFX та SceneBuilder.

Для додавання нового сувеніра введіть необхідні данні у відведені для цього поля та натисніть "Додати".
Для редагування данних двічі натисніть на обране поле, введіть необхідну інформацію та натисніть ENTER.
Щоб переглянути всі сувеніри натисніть "Показати всі".
Для збереження даних у файл натисніть "Зберегти".

Формат введення даних для завдання "Вивести інформацію про виробників заданого сувеніру, виробленого у заданому року" : "NAME 0000"
Формат введення та редагування дати: "00.00.0000"

Зчитування та запис інформації у файл реалізовано у класі IO. При відсутності необхідного файла створюється новий файл
з невеликим списком сувенірів для огляду можливостей програми.

Для сувенірів створено клас Souvenir для збереження назви, реквізитів виробника, дати випуску та ціни.
Для випрбників створено клас Manufacturer для збереження назви та країни.


Такі можливості:

• Вивести інформацію про сувеніри заданого виробника.
• Вивести інформацію про сувеніри, виготовлені в заданій країні.
• Вивести інформацію про виробників, чиї ціни на сувеніри менше заданої.
• Вивести інформацію по всіх виробниках та, для кожного виробника вивести інформацію про всі сувеніри, які він виробляє.
• Вивести інформацію про виробників заданого сувеніру, виробленого у заданому року.
• Для кожного року вивести список сувенірів, зроблених цього року.
• Видалити заданого виробника та його сувеніри

реалізовано у файлі SouvenirsProcessor. Для збереження данних використано List<>, ArrayList<> та Streams для обробки.

SouvenirsController - це клас адаптер який реалізує зв'язок між кодом та інтерфейсом та ініціалізує програму.
SouvenirsApplication - клас, що запускає програму.

SouvenirsProcessorTest - клас, який через JUnit перевіряє корректність виведення данних деяких методів із класа SouvenirsProcessor.
